Create and add quizzes for your audience through your WordPress blog via a simple interface.

== Description ==

Use this to add quizzes to your WordPress blog post using a simple interface.

**Features Include:**

* Customizable Fonts
* Multiple Choice Questions
* True/False Questions
* Scoring


If you have suggestions, feel free to email me at wpquizpro@gmail.com

== Installation ==

1. Upload `quizpro.php` and all the other files to the `/wp-content/plugins/quiz-pro` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a quiz in the settings.
4. Post it using the shortcode.

== Frequently asked questions ==

Q1. How do you add a quiz? 

* Go to "Create a Quiz"
* Select a quiz number and click Update
* Select the number of questions.
* Enter your questions and choose the correct answers.
* Save changes
* Add the quiz to your post using the shortcode [QuizPro quiz="xx"] where xx is your quiz number.


Q2. What types of questions are supported?

* Multiple choice questions with single answers
* True/False type questions
